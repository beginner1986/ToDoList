# ToDoList
Web app with Java Spring, React JS and MySQL database

## Server
Server app made with Java 11 and Spring Boot 2.6.3.
Initializers used:
- Lombok
- DevTools
- Test
- Web
- Data JPA
- H2 database
- JUnit

## Client
Client app made with JavaScript and React 17.02.
Packages used:
- jest
- axios
- use-axios-client
- bootstrap
- web-vitals
